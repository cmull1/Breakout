# Breakout
A simple clone of Atari Breakout using Python 3 and Pygame

## Controls
|     Action       |       Key       |
| --------------   | --------------- |
| Move bat left    | Left Arrow key  |
| Move bat right   | Right Arrow key |
| Play ball        | Up Arrow Key    |
| Pause game       | Enter/Return    |
|Return Ball to bat| r               |
